Project for the THI UXD Master VR/AR course

**VR GAME: First-Person Pacman**
- Horror / Atmosphärisch
	- Halbdunkel
	- Flackernde Lichter
	- Nebel/Rauch

- Time-Freeze (PickUp das Gegner einfriert und (kurzzeitig) Flucht ermöglicht)
